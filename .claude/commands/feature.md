# Feature Implementation

Implement a new feature based on the provided specification.

## Variables

spec_file: $1

## Instructions

1. Read the spec file to understand requirements
2. Analyze the codebase to understand current structure
3. Plan implementation steps
4. Implement the feature following the spec
5. Run tests to verify
6. Commit changes

## Run

1. Read spec: `cat $spec_file`
2. Check current branch: `git branch`
3. Implement according to spec
4. Run tests: `pnpm test`
5. Run lint: `pnpm lint`

## Report

- Summary of changes made
- Files modified/created
- Test results
- Any issues encountered
