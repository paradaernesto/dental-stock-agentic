# Chore Implementation

Execute a chore task (maintenance, config, tooling).

## Variables

task_description: $1

## Instructions

1. Understand the chore requirements
2. Identify files that need changes
3. Make minimal, focused changes
4. Test that the system still works
5. Commit changes

## Run

1. Analyze current state
2. Implement changes
3. Verify: `pnpm test && pnpm build`
4. Stage and commit

## Report

- What was done
- Files changed
- Verification results
