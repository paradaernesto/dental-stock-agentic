# Bug Fix

Fix a reported bug.

## Variables

bug_description: $1

## Instructions

1. Understand the bug from description
2. Reproduce the bug if possible
3. Identify root cause
4. Implement minimal fix
5. Add/regression test if applicable
6. Verify fix works
7. Commit changes

## Run

1. Investigate the bug
2. Implement fix
3. Run tests: `pnpm test`
4. Verify fix manually if needed
5. Commit

## Report

- Root cause identified
- Fix applied
- Tests passing
- Any follow-up needed
