# Implement from Spec

Implement a bounded task from a specification.

## Variables

spec_file: $1
task_id: $2 (optional)

## Instructions

1. Read and understand the spec
2. Identify scope of this implementation
3. Make focused, minimal changes
4. Ensure tests pass
5. Commit with reference to task_id if provided

## Run

1. Read spec file
2. Implement changes
3. Run tests
4. Commit: `git commit -m "${task_id}: implementation details"`

## Report

- Implementation summary
- Files changed
- Test results
